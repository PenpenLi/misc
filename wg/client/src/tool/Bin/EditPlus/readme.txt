[安装]
在EditPlus中依次点击[工具(T)]->[配置用户工具(C)]->[添加(D)]->[应用程序]
然后设置如下：
菜单文本：zencoding
命令：WScript.exe "D:\editPlus\zencoding\zencoding.wsf" 
参数：$(Prompt)
勾选[运行为文本滤镜]，选择[插入]
然后设置一个自己喜欢的快捷键吧。

[使用]
输入前按下之前设置的快捷键，然后在弹窗里输入快捷代码。

注：如果喜欢v1.0的选取片段方式，可以修改设置：
参数：$(CurSel)
运行为文本滤镜：[替换]
